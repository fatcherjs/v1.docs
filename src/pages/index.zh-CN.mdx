# 指引

fatcher 是一个轻量的 HTTP 请求库, 它基于 `fetch`，它允许我们在浏览器与 NodeJS 环境中使用原生的`fetch`来进行网络请求。

因为它是使用原生的 fetch 进行封装，所以我们在使用的时候，要求浏览器或者是 NodeJS 中要支持 fetch。

-   在现代浏览器中，对 fetch 的支持已经很好了，所以我们不用担心 fetch 的兼容性
-   在 NodeJS 中，则从`18.0.0`开始，对 fetch 已经有一定的支持

fatcher 目的是在于拥抱标准库的 fetch 的同时，提供 fetch 中一些没有提供的功能，以及让功能更好的拓展与复用。
